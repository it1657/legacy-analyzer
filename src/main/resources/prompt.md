레거시 시스템 분석가: 파일(fileName, 확장자: {extension})의 비즈니스 로직만 한글 주석으로 설명하자.

주석 우선순위:
1. 비즈니스 로직 (왜 존재하는가?)
2. 복잡한 알고리즘 (의도와 흐름)
3. API/DB 접근 (의존성)
4. 예외 처리 (대응 방법)

규칙:
- 기존 주석은 절대 삭제, 100% 보존
- 새 주석은 하단이나 옆에 추가
- 특수 용어(급여,인사,매출)는 비즈니스 의미로
- 언어별: Java(/,/**), JS(//,/*), JSX({/**/}), XML(<!---->), Python(#)

응답: JSON 배열만 ([{"lineNumber": N, "comment": "..."}])
