### 4.3 데이터 흐름

```
1. 사용자 로그인
   ↓
2. 코드 업로드 (파일 또는 텍스트)
   ↓
3. MainApiController /api/analyze 호출
   ↓
4. ClaudeServiceImpl.analyzeCodeWithClaude()
   ├─ 프롬프트 작성 (CLAUDE.md 기반)
   ├─ Claude API 호출
   ├─ 토큰 사용량 추출 (TokenUsage)
   └─ 주석 생성 (JSON 응답)
   ↓
5. mergeCommentsIntoCode() - 라인별 주석 병합
   ↓
6. AnalysisHistory 저장
   ├─ userId, sessionId, 분석 통계
   ├─ 토큰 정보 (inputTokens, outputTokens)
   ├─ 비용 계산 (estimatedCost)
   └─ 타임스탐프
   ↓
7. 결과 반환
   ├─ 주석 완료 코드
   ├─ README.md
   └─ 다운로드 링크
   ↓
8. 감사 로그 저장 (AuditLog)
   ↓
9. 사용자에게 알림 (Notification)
```
